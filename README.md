## Deepstream Demos

### Prerequisites
Register NGC API Key at https://ngc.nvidia.com/

Log in nvcr.io
```
docker login nvcr.io
```

Pull DeepStream image
```
docker pull nvcr.io/nvidia/deepstream:5.1-21.02-triton
```

Create DeepStream
